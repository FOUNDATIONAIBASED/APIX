# Automation (Mail / Telegram / SMS)

The web console exposes:

- **Mail** — IMAP, local index, SMS → email triggers  
- **Telegram** — bot settings (Settings), SMS → Telegram, IMAP → Telegram  
- **SMS** — inbound SMS forward, IMAP → SMS  

Plan features may gate sections; see **Plans** in the UI.
